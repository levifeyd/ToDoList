======Для запуска проекта=======
1. composer install
2. npm install
3.npm run build
3. npm run dev
4. создать файл .env и прописать правила для своей базы данных
5. php artisan migrate
6. Сделать следующие миграции
7. Загрузить сиды 
    php artisan db:seed --class=RoleSeeder
7. php artisan db:seed --class=UserSeeder
8. php artisan storage:link -  для ссылки на папку /storage/app/public/images/
